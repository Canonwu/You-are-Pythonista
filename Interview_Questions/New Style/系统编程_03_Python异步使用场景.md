# 系统编程_03_Python异步使用场景

## Question
下列哪个不是 Python 异步使用场景（）？

%!A. 没有时序上的严格关系!%

%!B. 不可以通过其他方式控制原子性的操作!%

%!C. 不影响主线程逻辑!%

%!D. 常用于IO操作等耗时操作，因为比较影响客户体验和使用性能!%

----

## Answer
@!B!@

----

## Analysis

异步的使用场景:

1、 不涉及共享资源，获对共享资源只读，即非互斥操作

2、 没有时序上的严格关系

3、 不需要原子操作，或可以通过其他方式控制原子性

4、 常用于IO操作等耗时操作，因为比较影响客户体验和使用性能

5、 不影响主线程逻辑
